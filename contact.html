お問い合わせ（改善案）

「ガイダンスページ → 目的別入口 → プログレッシブフォーム」の3段構成

SECTION 01

まず、ご相談の目的をお選びください

📐

概算を知りたい

用途・面積を伝えるだけで最短翌営業日に概算をお伝えします

💬

計画の相談をしたい

構想段階・土地未定でもOK。建築のプロに無料で相談

📘

資料を請求したい

→ カタログDLページへ

改善前はいきなりフォームが表示されていた。
改善案ではまず「何がしたいか」をガイダンスで選ばせる。これにより、ユーザーが「自分のやりたいことに対応してくれる」と安心でき、フォーム到達率が上がる。
また資料請求はカタログDLページに振り分け、問い合わせフォームへの流入を「相談意向あり」のユーザーに絞る。

SECTION 02

お問い合わせフォーム（プログレッシブ形式）

※ 以下のフォームに入力いただくと、より的確なご案内が可能です

STEP 1 — 基本情報

会社名 [必須]
ご担当者名 [必須]
メールアドレス [必須]
電話番号 [必須]

STEP 2 — 計画情報（任意・分かる範囲で）

用途 [選択]：倉庫 / 工場 / 店舗 / 事務所 / 危険物 / その他
希望延床面積 [選択]：150坪〜 / 300坪〜 / 500坪〜 / 未定
建設予定地の有無 [選択]：土地あり / 土地探し中 / 未定
希望時期 [選択]：半年以内 / 1年以内 / 1年以上先 / 未定
危険物の有無 [選択]：あり / なし / 分からない
概算ご予算 [選択]：3,000万円〜 / 5,000万円〜 / 1億円〜 / 未定
種別 [選択]：新築 / 建替え / 増築

STEP 3 — ご相談内容

自由記述欄（現在お困りのこと、確認したいこと等）
送信する
フォームをSTEP形式にし、STEP2は全項目「任意」のプログレッシブフィールドにする。
「分かる範囲でOK」というメッセージで記入ハードルを下げつつ、記入してくれた場合は初回ヒアリングが効率化する。

追加したフィールド（改善前にはなかったもの）：
・用途：概算精度が上がる
・延床面積：対象案件かの判断材料
・土地の有無：初回対応が変わる
・希望時期：優先度の判断
・危険物有無：対応フローが変わる
・概算予算：提案の方向性が変わる
・新築/建替え/増築：調査項目が変わる
